¿Qué es el diseño web?

El diseño web viene a ser todo lo que abarca el desarrollar interfaces digitales, ya sean diseños de sitios y/o paginas, como lo pueden ser tambien aplicaiones para web.

Hay una serie de principios basicos que se deben tener en cuenta a la hora de elaborar un diseño, algunos de estos principios son:

Visibilidad

Le permite al usuario tener una facil lectura de las funciones que se encuentran en el sitio web, de cómo funcionan y qué es lo que hacen.

  • Usabilidad y capacidad de aprendizaje continuo.

Ejemplos.

  • Boton de redactar en la aplicacion web de Gmail.
  • Barra superior con las funciones mas utilizadas de las aplicaciones de Office como lo son Word, Excel, Powerpoint, etc.
  • Uso de "tab bar" en dispositivos moviles.

Retroalimentación

Se define como la capacidad de mostrarle y/o notificarle al usuario cuando se haya ejecutado una accion dentro del sitio web.

  • Las formas de retroalimentación pueden ser visuales, verbales, auditivas y tactiles.

Ejemplos.

  • Cuando la papelera de reciclaje se encuentra vacia y esta cambia su icono al agregar archivos que han sido eliminados.
  • En los mandos de las consolas de videojuegos estos cuentan con vibraciones que se activan de forma unica al efectuar alguna accion dentro de un videojuego.
  • Por otro lado esta la alerta auditiva que ofrecen los sistemas de gps como Google Maps o Waze que indican cuando realizar giros, cambios de carril, entre otras cosas.

Restricciones

Viene a ser la forma de determinar el cómo restringir el tipo de interacción del usuario que puede tener lugar en un momento dado.

  • Estas pueden ser fisicas, culturales, semanticas y logicas.

Ejemplos.

  • Preguntar con anterioridad si se desea guardar lo que se estaba realizando cuando se intenta hacer un cierre forzozo en alguna aplicacion.
  • En Youtube al intentar hacer un comentario, este nos pedira que iniciemos o creemos una cuenta para poder realizar dicha accion.
  • En algunas paginas se pide seleccionar la ubicacion actual para ofrecer los productos disponibles dentro del area de cobertura.
